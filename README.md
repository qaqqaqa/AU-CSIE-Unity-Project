## 🛠️ 開發環境
* **Unity Version**: Unity6000.0.60f1 
* **語言**: C#
* **平台**: PC

## 🌟 核心功能與技術
* **[技術亮點 1]**: 自製火源擴散演算法，模擬真實火災場景。
* **[技術亮點 2]**: 無限容量滅火器邏輯實作。
* **[技術亮點 3]**:自定義 UI 系統與關卡切換機制（包含教學關卡與計時挑戰）。

## 📂 資料夾結構說明
* **Assets/**: 包含所有的 Scripts、Prefabs、Materials 及場景檔案。
* **ProjectSettings/**: Unity 專案的環境設定。
